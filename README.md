# capcom — moved

The **capcom** engine kernel now lives in the
**[autarchy-ai/aphelion](https://github.com/autarchy-ai/aphelion)** repository,
under [`kernel/`](https://github.com/autarchy-ai/aphelion/tree/dev/kernel)
(the `capcom` Cargo workspace). See
[ADR-017](https://github.com/autarchy-ai/aphelion/blob/dev/docs/adrs/ADR-017.md)
for why the kernel and the Java 21 product now share one repository.

This repository is **archived** and read-only. Its history is preserved here
and was imported into `aphelion/kernel/` via `git subtree`. New work — issues,
pull requests, requirements — happens in
[`autarchy-ai/aphelion`](https://github.com/autarchy-ai/aphelion).
